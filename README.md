# sn.crawl.news 
Crawls stock news and publishes those articles.

## Getting Started
Install the module with: `npm install crawl.news`

## Messages Published
  - sn.news.article.create

## Worker Messages Subscribed To
  - sn.picker.symbol.create
  - sn.picker.symbol.delete
